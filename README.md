# Breaking Bad App
iOS application that shows list of episodies of breaking bad.

## API that I was using:
https://breakingbadapi.com/api/characters

### Technologies && tools:

- MVVM
- RxSwift
- Alamofire
- Decodable
- URLSession
- Creational Pattern Builder
- Protocols
- Views Programmatically
- UIKit
